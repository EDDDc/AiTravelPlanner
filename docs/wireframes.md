# 低保真原型草稿

> 本文档用于描述 Web 端主要界面的低保真原型，帮助团队在开发前统一交互预期。若后期产出设计稿，可在此基础上补充或替换。

## 1. 仪表盘 / 行程列表

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶部导航：Logo | 计划筛选 | 新建行程按钮 | 用户菜单                    │
├──────────────────────────────────────────────────────────────────────┤
│ 左侧侧栏                                                                │
│ ┌────────┐  ┌────────────────────────────────────────────────────┐ │
│ │ 行程概览 │  │ 欢迎语 + 快速提示                                      │ │
│ │ 我的行程 │  │ 当前行程卡片（目的地、日期、预算）+ 快捷操作           │ │
│ │ 地图视图 │  │ 最近更新的行程列表                                     │ │
│ │ 预算中心 │  │ 待办事项 / 提醒                                        │ │
│ │ 设置中心 │  │                                                        │ │
│ └────────┘  └────────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────────────┘
```

核心要素：

- 侧边栏固定全局导航。
- 主区展示当前选中计划以及其它行程列表。
- 顶部按钮提供新建行程入口。

## 2. 新建行程弹窗 / 步骤页

```
┌──────────────────────────新建行程───────────────────────────┐
│ 步骤条：1. 基础信息  >  2. 偏好设置  >  3. 语音补充（可选） │
├────────────────────────────────────────────────────────────┤
│ [基础信息]                                                   │
│  目的地输入框 | 起止日期选择 | 预算输入 | 人数/同行类型        │
│  偏好标签（美食/文化/亲子/户外等）                            │
│  语音录制按钮（可直接录入需求）                               │
│                                                                │
│ 底部：取消 | 上一步 | 下一步                                  │
└────────────────────────────────────────────────────────────┘
```

- 语音录制按钮需显示录制状态与识别文本。
- 录音完成可编辑识别结果。

## 3. 行程详情（卡片视图 + 时间轴）

```
┌──────────────────────────────────────────────────────────────┐
│ 顶部：行程标题 | 日期范围 | 预算概览 | 操作按钮（导出/复制）       │
├─────────────────┬────────────────────────────────────────────┤
│ 左侧：时间轴      │ 右侧：每日行程卡片                               │
│ ┌────────────┐ │ ┌────────────────────────────────────────┐ │
│ │ Day 1  日期   │ │ [交通] 08:00 - 10:00 机场 → 酒店              │ │
│ │ Day 2         │ │ [景点] 11:00 - 13:00 浅草寺 + 午餐            │ │
│ │ Day 3         │ │ [备注输入框]                                   │ │
│ └────────────┘ │ │ 拖拽手柄 | “添加活动”按钮                      │ │
│                 │ └────────────────────────────────────────┘ │
├─────────────────┴────────────────────────────────────────────┤
│ 底部：语音记账入口 | 行程实时提醒开关 | “切换地图视图”按钮            │
└──────────────────────────────────────────────────────────────┘
```

- 右侧卡片支持拖拽排序、修改时间段、添加自定义活动。
- 左侧时间轴点击后主面板滚动到对应日期。

## 4. 地图视图

```
┌───────────────┬──────────────────────────────────────────────┐
│ 左侧行程面板     │ 右侧地图（高德）                               │
│ ┌───────────┐ │ ┌────────────────────────────────────────┐ │
│ │ Day 1 活动列表 │ │ 地图标记：编号 + 类型图标                   │ │
│ │ Day 2 活动列表 │ │ 下方浮层显示选中景点详情 + 导航按钮         │ │
│ └───────────┘ │ └────────────────────────────────────────┘ │
└───────────────┴──────────────────────────────────────────────┘
```

- 活动列表点击后地图定位并展示路线。
- 地图右上角提供筛选（交通/景点/餐饮）和视图切换。

## 5. 预算中心

```
┌──────────────────────────────────────────────────────────────┐
│ 顶部：预算概览卡片（总预算 / 已用 / 剩余）                     │
├─────────────────┬────────────────────────────────────────────┤
│ 左侧分类图表      │ 右侧记账明细列表                                │
│ 圆环图 / 条形图    │ 表格：日期 | 金额 | 分类 | 来源(语音/手动) | 备注 │
│                   │ 列表顶部提供“语音记账”“导出 CSV”按钮             │
├──────────────────────────────────────────────────────────────┤
│ 底部：预算提醒配置（阈值、通知方式）                               │
└──────────────────────────────────────────────────────────────┘
```

## 6. 设置中心（API Key 输入）

```
┌──────────────────────────────────────────────────────────────┐
│ Tab：个人信息 | API Key | 通知设置                              │
├──────────────────────────────────────────────────────────────┤
│ [API Key]                                                     │
│ 阿里云百炼 Key  输入框  [测试连接] 按钮                          │
│ 科大讯飞 Key   输入框组（AppID / APIKey / APISecret）           │
│ 高德地图 Key   输入框  [测试连接]                               │
│ 说明文本：不会存储在浏览器本地，可随时更新                      │
│ 底部提交按钮 + 上次更新人/时间展示                               │
└──────────────────────────────────────────────────────────────┘
```
